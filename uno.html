¿QUÉ ES HTML?

HTML es el acrónimo de HyperText Markup Language, que en español significa "Lenguaje de Marcado de Hipertexto". Es un lenguaje de programación utilizado para crear y estructurar el contenido de las páginas web. HTML utiliza etiquetas para definir elementos como encabezados, párrafos, enlaces, imágenes y otros elementos que se muestran en un navegador web. Además, también se utiliza en conjunto con otros lenguajes de programación como CSS y JavaScript para crear páginas web interactivas y dinámicas.



elementos de HTML

Los elementos HTML son las etiquetas que se utilizan para definir la estructura y el contenido de una página web.

  • !DOCTYPE html : Define el tipo de documento HTML que se está utilizando.
  • html : Define el elemento raíz de un documento HTML.
  • head: Define la sección de encabezado de un documento HTML y contiene información como el título de la página, los metadatos y los enlaces a archivos CSS y JavaScript.
  • body: Define la sección de contenido de un documento HTML y contiene todos los elementos que se muestran en la página, como texto, imágenes, videos, formularios y otros elementos HTML.
  • h1 - h6: Define los encabezados de la página, donde h1 es el encabezado más importante y h6 es el encabezado menos importante.
  • p : Define un párrafo de texto.
